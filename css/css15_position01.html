Static is the default value and not affected by the top, bottom, left, and right properties.

Relative behaves the same s static unless you add some extra properties.

Setting the top, right, bottom, and left properties of a relatively-positioned element will cause it to be adjusted away from its normal position.

A fixed element is positioned relative ti the viewport, which means it always stays in the same place even if the page is scrolled. the top, right, bottom, and left properties are used to position the element.

Absolute behaves like fixed except relative to the nearest positioned ancestor instead of relative to the viewport.

If an absolutely-postitioned element has no positioned ancestors, it used the document body, and still moves along with page scrolling.
